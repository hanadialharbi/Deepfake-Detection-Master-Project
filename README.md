## 🧠 Overview
This project presents a deepfake detection system designed for real-world video analysis.  
The proposed framework combines a hybrid CNN–TCN model with federated learning to ensure privacy, scalability, and robustness.

---

## 🚀 Features
- Hybrid CNN–TCN architecture  
- Federated Learning (FedAvg)  
- Real-world dataset  
- Explainable results (Grad-CAM)  
- Interactive prototype
